# Hello World Vercel Demo

This is a sample project to demonstrate a clean folder structure.

## Folder Structure
- `src/`: Source code (HTML, JS, CSS).
- `public/`: Static assets (icons, robots.txt).
- `tests/`: Project tests.
- `src/components/`: Reusable logic/UI components.
- `src/assets/`: Global styles and images.

## Running Locally
Simply open `src/index.html` in your browser.
